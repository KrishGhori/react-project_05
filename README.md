# React Project five

A modern React application built using Vite. This project demonstrates component-based architecture, state management, and clean project structure.

---

##  Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS
- ESLint


##  Features

- Built with React and Vite  
- Fast development server with Hot Module Replacement  
- Functional components using JSX  
- State management using useState  
- State lifting between components  
- Reusable component structure  
- Organized folder architecture  
- ESLint integration for clean code  

---
## How It Works

-App.jsx stores the state using useState.
-The state and setter function are passed to Card components as props.
-Both Card components share the same state.
-Updating input in one card updates both cards.
